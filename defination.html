Define the terms: Website, Webpage, Web browser, Web server, HTML,CSS.

  1. Web Site
    • Website is a collection of web pages.
  2. Web Page
    • Webpage is a collection of Data.
  3. Web Browser
    • webbrowser is a collection of websites.
  4. Web server
    • A web server is computer software and underlying hardware that accepts requests via HTTP or its secure variant HTTPS.
  5. HTML
    • The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.
  6. CSS
    • Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML.
